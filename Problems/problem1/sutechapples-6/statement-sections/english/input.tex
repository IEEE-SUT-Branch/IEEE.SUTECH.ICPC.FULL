
The first line of the input contains two space-separated integers $N$ and $X$ ($1 \le N \le 100$, $1 \le X \le 100$) --- the number of students and the maximum allowed apples per student, respectively.
The second line contains $N$ space-separated integers $A_1, A_2, \dots, A_N$ ($1 \le A_i \le 100$) --- where $A_i$ is the number of apples requested by the $i$-th student.

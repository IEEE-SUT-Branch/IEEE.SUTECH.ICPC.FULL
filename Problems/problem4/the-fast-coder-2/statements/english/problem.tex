\begin{problem}{The Fast Coder}{standard input}{standard output}{1 second}{256 megabytes}

Rami is practicing for the placement test. There are $N$ practice problems arranged in a straight line. Solving the $i$-th problem takes exactly $T_i$ minutes. Rami has only $M$ minutes of free time.
Because Rami is systematic, he wants to choose a contiguous subsegment of problems to solve (i.e., problems from index $L$ to index $R$). What is the maximum number of problems Rami can solve within his $M$ minutes?

\InputFile
The first line contains two integers $N$ and $M$ ($1 \le N \le 10^5$, $1 \le M \le 10^9$).
The second line contains $N$ integers $T_1, T_2, \dots, T_N$ ($1 \le T_i \le 10^4$).

\OutputFile
Print the maximum number of contiguous problems Rami can solve.

\Example

\begin{example}

\Note
This is a classic Sliding Window / Two Pointers problem. Maintain a window [L, R] and a current sum. If the sum exceeds $M$, increment $L$. Record the maximum length $R - L + 1$. Time Complexity $O(N)$.

\end{problem}

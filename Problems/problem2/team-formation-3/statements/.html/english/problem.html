Team Formation
time limit per test
1 second
memory limit per test
256 megabytes
input
standard input
output
standard output

Arrays, Sorting, Greedy.

In the ECPC, a team consists of exactly 3 members. You have $$$N$$$ students, and each student has a programming skill level $$$S_i$$$. You want to form exactly one team of 3 members such that the total sum of their skill levels is as maximum as possible.

Input

The first line contains an integer $$$N$$$ ($$$3 \le N \le 10^5$$$) — the number of available students. The second line contains $$$N$$$ integers $$$S_1, S_2, \dots, S_N$$$ ($$$1 \le S_i \le 10^4$$$) — the skill level of each student.

Output

Print a single integer representing the maximum possible skill sum of a 3-member team.

Example

Input
5
10 20 5 30 15
Output
65

Note

Sort the array in descending order and sum the first 3 elements. Time Complexity

A. Maximum Subarray Sum

schedule Time limit: 1.0s memory Memory limit: 256MB

Given an array of $n$ integers, your task is to find the maximum sum of values in a contiguous, non-empty subarray.

Input

The first input line has an integer $n$: the size of the array. The second line has $n$ integers $x_1, x_2, \dots, x_n$: the contents of the array.

Output

Print one integer: the maximum subarray sum.

Constraints

  • $1 \le n \le 2 \cdot 10^5$
  • $-10^9 \le x_i \le 10^9$
Sample Input 
8
-1 3 -2 5 3 -5 2 2
Sample Output 
9
expand_more
123456789101112131415

// Problem A: Maximum Subarray Sum

#include <iostream>

#include <vector>

#include <algorithm>

using namespace std;

int main() {

int n;

cin >> n;

vector<long long> x(n);

for (int i = 0; i < n; ++i) cin >> x[i];

long long ms = -1e18, cur = 0;

for (auto i : x) {

cur = max(i, cur + i);

ms = max(ms, cur);

}

cout << ms << endl;

return 0;

}
